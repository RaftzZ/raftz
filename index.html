Sobre mim

Comecei pensando em seguir uma carreira puxada para o design gráfico, mas com uma maior procura de devs no mercado, decidi seguir como dev depois de algumas pesquisas.


Com a ampliação dos meios de tecnologia, a questão da programação de um aplicativo ou de uma site virou uma profissão real com o mundo digital, onde muitos querem que seus produtos sejam bem vistos pelas grandes massas.

Serviços

Posso oferecer serviços de como programação Front-end e Back-end como dev full stack, mas também ofereço serviços como designer gráfico.

Planejamento

Teremos uma base de planejamento para tudo que irá envolver o produto, para que fique tudo melhor para o cliente.

Ampliação Criativa

Faremos alguns testes e modificações para conseguir chegar a uma ampliação criativa.
